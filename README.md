# Subliminal Podcast — Do Nice Things

A 5-page podcast website built for the BIWD2210 Web Technology assignment.

## Pages
1. `index.html` — Home page with hero, featured episode, episode cards, host preview
2. `episodes.html` — Full episode list with all 12 episodes
3. `episode-detail.html` — Episode detail page with audio player, show notes, video embed
4. `hosts.html` — Host profiles with bios and social links
5. `contact.html` — Contact form with FAQ accordion

## How to Open
1. Open the `subliminal-podcast` folder in Sublime Text
2. Right-click `index.html` → Open in Browser
   OR install the "Open in Browser" Sublime package

## Hosting on GitHub Pages
1. Create a new GitHub repository
2. Upload all files (keep folder structure)
3. Go to Settings → Pages → Source: main branch
4. Your site will be live at: https://yourusername.github.io/subliminal-podcast

## Features
- Semantic HTML5 throughout
- External CSS stylesheet (style.css)
- Flexbox & CSS Grid layouts
- Responsive mobile design
- Embedded YouTube video (episode-detail.html)
- HTML5 audio player
- Animated ticker tape
- CSS animations & hover states
- Ordered & unordered lists
- Internal navigation linking all 5 pages
- External hyperlinks (WHO, Headspace, etc.)
- Image gallery with captions
- Contact form with validation
- FAQ accordion (details/summary)
- Noise texture overlay
- Google Fonts (Bebas Neue, Playfair Display, DM Sans)
